Let's make a shooter game!

first we should call the init() function to initialize our game:

init({
	width: 640,
	height: 480,
});

this will help us create a 640x480 canvas for drawing the game view. Try running this piece of code! Wild 640x480 canvas appears!

then let's load our resources!

init({
	width: 640,
	height: 480,
});

loadSprite("guy", "guy.png");

this will load the image file "guy.png" into the asset manager, with the name "guy"

Let's put this image into our game as a character! To do this we are going to create a new scene, called "main"

init({
	width: 640,
	height: 480,
});

loadSprite("guy", "guy.png");

scene("main");

// put stuff in "main" scene here

start();

any code below this scene() call will be describing what we're going to put into this scene, until another scene() call, or a start() call to end the scene building and start the game

Let's add the player sprite to the scene:

init({
	width: 640,
	height: 480,
});

loadSprite("guy", "guy.png");

scene("main");
const player = sprite("guy");
start();

the function sprite(name) adds a sprite with specified name to the scene, returning the reference to the game object. Try run the game now! There'll now be a player in the middle of the screen!

now let's make the player move around with our keys!

init({
	width: 640,
	height: 480,
});

loadSprite("guy", "guy.png");

scene("main");
const player = sprite("guy");
const speed = 320;

keyDown("left", () => {
	player.move(vec2(-speed, 0));
});

keyDown("right", () => {
	player.move(vec2(speed, 0));
});

keyDown("up", () => {
	player.move(vec2(0, speed));
});

keyDown("right", () => {
	player.move(vec2(0, -speed));
});
start();

keyDown() will register a keyDown event, so every frame user is holding the specified key down, the callback would run. player.move() does exactly what it says, it moves the player. Try running the game now, the player will move when you hold down arrow keys
